#Meteor Simulator

#Description
This application simulates a meteor flying and crashing show his impact in red.In addition the person can add
more meteors to do custom simulations.

Design Patterns:
DAO(Data Access Object):Seperating Main code from Database operations
MVC(Model,View,Controller):Structure of our program
Singleton:Only one controller can be created
Service:Seperator between DAO and MVC so controller does not have a lot of code.
